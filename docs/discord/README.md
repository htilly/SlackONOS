## Discord App Directory Benefits

- ✅ **No OAuth required** - Simple invite link
- ✅ **No public URL needed** - Gateway WebSocket connection
- ✅ **Self-hosted friendly** - Perfect for firewall-protected deployments
- ✅ **Simpler than Slack** - No multi-workspace complexity
